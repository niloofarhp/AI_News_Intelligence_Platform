# AI News Intelligence Platform

An AI-powered system for intelligent news discovery, analysis, and personalization.  
This platform integrates **semantic search, Retrieval-Augmented Generation (RAG), LLM-based summarization, and recommendation systems** to help users explore and understand large collections of news articles.

The system demonstrates how modern **Natural Language Processing (NLP) and Large Language Models (LLMs)** can be used to build scalable AI-driven content platforms.

---

## Key Features

- **Semantic News Search** using sentence-transformer embeddings and FAISS vector search  
- **RAG-based Question Answering** powered by large language models  
- **Automatic Article Summarization** using LLMs  
- **Personalized News Recommendations** based on user search history  
- **Interactive Web Interface** built with Streamlit
